# Noobish-Chat-Room

### One line install 
```bash
 
 git clone https://github.com/adilrahman/Noobish-Chat-Room.git && cd Noobish-Chat-Room && chmod +x setup.sh && sudo ./setup.sh 

```

### Usage
> python -m SimpleHTTPServer <port>
> node server.js
> Vist : `http://host:port`
